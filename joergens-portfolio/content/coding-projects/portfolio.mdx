# Backend Infrastructure
    ## **AWS**
        ### *EC2:* virtual machine to host Rhino.Compute Server
        ### *Lambda:* serverless architecture to run python functions from JavaScript 
        ### *S3:* stores and generates signed URLs of Grasshopper definitions 
# Data Processing and Analysis
    ## **Rhino.Compute Server**
    ### i. computational service for the Rhino 3D modelling software
    ### ii. allows remote execution of Rhino scripts 
    ## **Grasshopper** (Rhino Plugin) 
    ### i. enables algorithmic modelling, generative design and parametric workflows
# Data Analysis and Visualization
    ## **Python**
        ### *NetworkX:* provides data structures for representing graphs and algorithms to analyze network properties
        ### *Matplotlib and NumPy:* data visualization and numerical computing
